JSONP

JSONP é um método para enviar dados JSON sem se preocupar com problemas entre domínios.
JSONP não usa o objeto.XMLHttpRequest
JSONP usa a tag em vez disso.<script>
Solicitar um arquivo de outro domínio pode causar problemas devido à diretiva entre domínios.
Solicitar um script externo de outro domínio não tem esse problema.
O JSONP usa essa vantagem e solicita arquivos usando a tag de script em vez do objeto XMLHttpRequest.
